# SCIM API and Trint
Centralize identity management using SCIM with Trint.
SCIM is used for Cross-domain Identity Management. SCIM API simplifies user identity management in cloud applications. It automates creating, updating, and deleting user accounts across systems, ensuring consistent and accurate user data while reducing administrative tasks.
